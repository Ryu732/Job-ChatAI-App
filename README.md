<br>

## アプリURL

- 会社検索機能

## 使用技術
| **カテゴリ**     | **技術**             | **説明** |
|------------------|----------------------|----------|
| フロントエンド   | Vue.js, Vuetify 3         | |
| バックエンド     | Node.js, Express              | |
| データベース     | MongoDB atlas             | DBをクラウド上で構築 |
|       API    | Bing Search API      | Web検索用API |
|        | Gemini API           | Geminiを使用した文章生成 |
| ライブラリ     | Langchain.js  | AIのプロンプト作成などに利用 |
